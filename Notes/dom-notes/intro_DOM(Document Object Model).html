Document Object Model

Document represents everything of the tree

Document.firstElementChild represent everything inside the html tag

Can store it as variable of js

var heading = document.firstElementChild.lastElementChild.firstElementChild
heading.innerHtml = "Good Bye"
heading.style.color = "red"

Using querySelector

Select the query tags from everything inside document

document.querySelector("Input").click()
