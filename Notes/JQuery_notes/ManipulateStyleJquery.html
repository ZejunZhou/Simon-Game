Manipulate Style with Jquery

$("h1").css("property", "propertyValue")
Example: $("h1").css("color", "red")
But directing change css sytle by js is not promoted Soluation: use classlist (add/remove/toggle) 
$("h1").addClass("Classname")
$("h1").removeClass("Classname")
Add mutiple class 
$("h1").addClass("Classname1 Classname2")
which simply seperate by space
Jquery can tell whether the element has the class or not 
$("h1").hasClass("Classname")
which return true or false
