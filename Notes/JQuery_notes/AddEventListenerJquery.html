Add Event Listner Jquery

$("h1").click(function(){
    $("h1").css("color", "purple")
})
  • h1 here is the target applied to
  • click here is the eventType
  • Anonymous function is the callback function
Jquery not need a for loop to add to all buttons 
$("button").click(function(){
    $("h1").css("color", "blue")
    })
will applied to All button
Add different event 
$("button").on("click", function(){
    $("h1").css("color", "blue")
    })
