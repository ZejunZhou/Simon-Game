Position

Static:HTML rule by default

Relative:Relative postition is relative to where the element would being displayed (static) if is just follows the natural HTML flow

Relative Example:

Right 20px means there is a 20px margin betwween its previous edge and current edge

Absolute:Adding the margin relative to its parent element coordinate

Fixed:Always fixed to the position relative to the body

Lay out

Left, right, top, bottom if it is relative it does not affect how other element display (it will overwriitn or cover

Left, right, top, bottom if it is absolute it will affect how other element display (no longer consider as a part of natural flow)
